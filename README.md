# CSES Problem Set Solutions

Solutions to problems from the [CSES Problem Set](https://cses.fi/problemset/) written in C++.

## Structure

| Folder | Problems |
|--------|----------|
| `introductory_problems/` | Basic warmup problems |
| `tree_algorithms/` | Tree traversal and path problems |
| `dynamic_programming/` | DP problems (knapsack, grid, string, digit) |
| `others/` | Miscellaneous / LeetCode scratch |
